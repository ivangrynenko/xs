INSTALLATION INSTRUCTIONS

Step 1: Upload the files to your WHMCS install

	1. Ftp to your WHMCS installation directory
	2. Upload tppwregistrar.php directory:
		/modules/registrars/tppwregistrar
	*If the directory does not exist, please create it*

Step 2: Configuring WHMCS

	1. Log in to your WHMCS admin
	2. Locate Setup > Products/Services > Domain Registrars
	3. Click "Activate" next to the  TPP Wholesale module
	4. Click "Configure" and enter in your API Credentials
	5. Save changes
